# Everand (Scribd) Book URLs Scraper
This scraper automates the process of collecting book URLs from Everand (Scribd) in various languages.

German (german)
Japanese (japanese)
Hindi (hindi)
Portuguese (portuguese)
Russian (russian)
Spanish (spanish)
French (french)
Chinese (chinese)
Arabic (arabic)
Afrikaans (afrikaans)
Turkish (turkish)
Swedish (swedish)
Polish (polish)
Indonesian (indonesian)
Italian (italian)
Greek (greek)
Danish (danish)
Dutch (dutch)
Catalan (catalan)
